SSD Virtual Battery has been deployed!

Get the application URL by running:
  kubectl port-forward svc/{{ include "ssd-virtual-battery.fullname" . }} 3000:{{ .Values.service.port }}
  Then visit: http://localhost:3000

To check the status of your pods:
  kubectl get pods -l app.kubernetes.io/instance={{ .Release.Name }}

To view logs:
  kubectl logs -l app.kubernetes.io/instance={{ .Release.Name }},app.kubernetes.io/component=web
  kubectl logs -l app.kubernetes.io/instance={{ .Release.Name }},app.kubernetes.io/component=worker

To run the data collection manually:
  kubectl exec -it deploy/{{ include "ssd-virtual-battery.fullname" . }} -- bin/rails virtual_battery:collect_data
